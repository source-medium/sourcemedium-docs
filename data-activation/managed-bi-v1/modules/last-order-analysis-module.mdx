#### What is the Last Order Analysis module and what data can be found there?
The Last Order Analysis populates data from your online store(s), subscription platform, and as designated in the Configuration Sheet. Customers are surfaced whose last order is within the dates selected in the date range filter and can be first time orders and/or repeat orders.
