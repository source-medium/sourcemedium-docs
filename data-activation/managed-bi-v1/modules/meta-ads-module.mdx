#### What is the Meta Ads module and what data can be found there?
   - Specialized module that has been optimized for Meta Ads. This module reports Meta data on the attribution window set at the campaign level.  
   - This is where you will find insights into your Meta ads as reported by Meta



#### Common Questions & Insights That Can Be Answered Here:
<AccordionGroup>
<Accordion title="Where can I find my Meta ads Revenue? CPO? CVR?">
These metrics can be found in the **Performance by Date** table.

</Accordion>


<Accordion title="Where can I find which of my Meta campaign types or specific campaigns are preforming best?">
This can be found in the **Performance by Date & Campaign** table and the **Performance by Date & Campaign Type** table


</Accordion>


</AccordionGroup>

#### Potential Reporting Differences & Discrepancies
SourceMedium reports Meta data on the attribution window set at the campaign level.  If different attribution windows are set between campaigns, this can cause confusion when validating vs. the Meta UI.
