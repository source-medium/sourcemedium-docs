## Follow this integration guide to connect your Klaviyo data to Source Medium.

### Requirements

- **Admin Access** to your Klaviyo account to be able to copy and share your Klaviyo API keys.
- Premium Integration

### Steps

1. Navigate to the **API Keys** section of your Klaviyo account to create an API key for Source Medium 
    1. Log into your Klaviyo account
    2. Click the user menu (top right corner of the page), then click **Account**
        
        ![](/images/article-imgs/klaviyo-integration/Untitled.png)
        
    3. On the **Account** page, click **Settings > API Keys**
        
        ![](/images/article-imgs/klaviyo-integration/Untitled1.png)
        
2. Create and label your API key for Source Medium 
    1. On the **API Keys** page, click the **Create API Key** button to create an API key
    2. In the **Label** column, click the pencil icon next to the API key you just created
    3. Enter **Source Medium integration** as the label for the API key
    4. Select the `Read-Only Key` and ensure all fields have been given Read-Only access
        
        ![](/images/article-imgs/klaviyo-integration/Untitled2.png)
        
    5. Click the Create button in the top right of the page
        
        ![](/images/article-imgs/klaviyo-integration/Untitled3.png)
        
    
3. Send the saved API key to **integrations@sourcemedium.com**
