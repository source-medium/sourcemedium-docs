## Follow this integration guide to connect your Mailchimp data to SourceMedium

### Requirements

- Admin access to your Mailchimp account

### Steps

1. Create a Mailchimp API key in your Mailchimp account.
2. Send the API key to **[integrations@sourcemedium.com](mailto:integrations@sourcemedium.com)**.

### Additional resources

- Mailchimp API key documentation: https://mailchimp.com/developer/marketing/guides/quick-start/
