## Follow this integration guide to connect your ShareASale data to Source Medium.

### Requirements

- **Admin access** for ShareASale account to view your API token and secret key

### Steps

1. Retrieve your API token and secret key 
    1. [Navigate to Merchant API](https://account.shareasale.com/m-apiips.cfm) > select **Require IP address match only for version 1.1 and lower** **>** **Update Settings**
    
    ![](/images/article-imgs/shareasale-integration/Untitled.png)
    
      b.  Note your **Token**, **API Secret** and **Merchant ID**
    
2. Email the **Token, API Secret, Merchant ID** credentials to integrations@sourcemedium.com
